* npm WARN enoent ENOENT: no such file or directory

这个问题可能是npm版本过低引起的。经过反复尝试，忽然解决了（可能需要重启操作系统）。可能有效的操作步骤如下：
```bash

> rm -rf node_modules/
> npm i -g npm
> node -v
v9.0.0
> npm -v
6.9.0
> npm cache clean --force
> npm install

```
