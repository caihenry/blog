# caihenry.github.io of HEXO blog tutorial
>  这是一个公开博客，主要是记录技术分享

## 安装

``` bash

npm install hexo-generator-sitemap --save
npm install hexo-generator-baidu-sitemap --save

```

## 使用方法

``` bash
# 创建文章
cd source/_post
python ../../tool/create <title>
编辑新生成的文件内容
```

## hexo命令

* hexo d -g (生成并部署)
